# Changelog

本项目的主要版本更新记录。

## v0.1.4 - 2026-05-08

- 新增根目录 `CHANGELOG.md`，集中整理版本说明。
- 同步扩展 `manifest.json` 版本号到 `0.1.4`，让浏览器扩展页显示与项目版本一致。
- 同步本地 helper 版本到 `0.1.4`，为后续发布保持一致。
- README 补充更新记录入口，并整理项目结构里的版本说明文件。

## v0.1.3 - 2026-05-08

- 新增 `亮度` / `透明` 悬浮按钮，鼠标移上去弹出垂直滑杆。
- 亮度支持实时调节，范围 `50% ~ 150%`。
- 透明度支持实时调节，范围 `20% ~ 100%`。
- 鼠标移出小窗后仍完全隐藏；鼠标移回后恢复到你设置的可见态透明度。
- 修复 README 中 Mermaid 图在 GitHub 上的渲染问题。

## v0.1.2 - 2026-05-06

- 支持关闭透明小窗后把播放进度同步回原网页视频。
- 支持再次打开透明小窗时从同步后的进度继续播放。
- 整理本地 helper 版本并发布 `v0.1.2`。

## v0.1.1 - 2026-05-03

- 鼠标移出透明小窗时自动暂停视频。
- 鼠标移回小窗时恢复由隐身模式自动暂停的视频。
- 添加 MIT License。

## v0.1.0 - 2026-05-03

- 初始版本发布。
- 支持通过浏览器扩展把 B 站视频交给本地 Electron 透明置顶小窗播放。
- 支持基础的透明隐身逻辑与跨应用置顶播放。
